如何获取 Bearer Token
  1. 登录 ads.moloco.com
  2. 按 F12 打开 DevTools
  3. 点 Network 标签页
  4. 页面任意操作触发请求
  5. 点任意 API 请求 → Headers
  6. 复制 Authorization: Bearer ... 后面的值
Token 有效期 16 小时，过期后重新复制即可
就绪 — 粘贴 token 后开始
